# ✨ Glow & Glam – Full Stack Beauty E-commerce Website

Glow & Glam is a modern full-stack beauty and skincare e-commerce website inspired by Nykaa. It provides users with a smooth online shopping experience, including product browsing, search, wishlist, cart, checkout, and order confirmation.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/GlowAndGlam.git
```

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

Open `index.html` using Live Server or any local web server.

---

## 🌐 API Endpoints

### Get All Products

```
GET /api/products
```

### Add Product

```
POST /api/products
```

### Get Product by ID

```
GET /api/products/:id
```
